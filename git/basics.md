# Git Basics

## Core idea

Git is a version control system. It helps track changes in files and keep a history of project versions.

## Basic workflow

```bash
git status
git add <file>
git commit -m "message"
git push
```

## Commands
- git status
 Shows the current state of repository.
- git add
 Prepares selected changes for the next commit.
- git push
 Uploads local commits to GitHub

## Notes
- git add - does not save changes permanently
- git commit - saves changes locally
- git push - sends commits to GitHub
- git status - should be used often
