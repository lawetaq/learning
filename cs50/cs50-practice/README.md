# CS50 C Practice

Small C programs written while learning CS50x without the CS50 library.

## Programs

- `hello.c` — prints hello world
- `name.c` — asks for a name using `fgets`
- `compare.c` — compares two integers
- `password.c` — simple password check using `strcmp`

## Build and run

```bash
make hello
./hello
make name
./name
make compare
./compare
make password
./password
```
