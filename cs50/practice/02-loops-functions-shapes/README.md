# 02 — Loops, Functions, and Shapes

C practice programs focused on loops, input validation, helper functions, nested loops, and terminal-based shape drawing.

## Programs

- `counter.c` — prints numbers from 1 to 10
- `meow.c` — prints `Meow!` multiple times with input validation
- `password_attempts.c` — password check with up to 3 attempts
- `meow_function.c` — meow program using a custom function
- `column.c` — prints a vertical column of `#`
- `row.c` — prints a horizontal row of `#`
- `square.c` — prints a square using nested loops
- `square_function.c` — prints a square using a helper function
- `rectangle.c` — prints a rectangle using height and width
- `rectangle_function.c` — prints a rectangle using a helper function
- `mario_stairs.c` — prints left or right Mario-style stairs
- `better_mario_stairs.c` — improved stairs program with helper functions
- `mario_pyramid.c` — CS50-style double pyramid with height validation from 1 to 8

## Concepts practiced

- `for` loops
- `do while` loops
- input validation
- `break`
- flag variables
- nested loops
- helper functions
- function parameters
- local scope
- logical OR `||`
- logical AND `&&`
- character input with `scanf(" %c", &dir)`
- terminal-based shape drawing

## Build and run

```bash
make mario_pyramid
./mario_pyramid
```
