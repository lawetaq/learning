# Algorithms Practice in C

Small C programs written to practice fundamental algorithms and problem-solving concepts.

## Topics Covered

- Linear Search
- Minimum Value Search
- Maximum Value Search
- Sum of Array Elements
- Selection Sort
- Bubble Sort
- Binary Search

## Goals

- Practice working with arrays
- Improve understanding of loops and conditions
- Learn how common algorithms work internally
- Build confidence writing algorithms from memory
- Debug and fix logical mistakes independently

## Files

- `linear_search.c`
- `minimum.c`
- `maximum.c`
- `sum.c`
- `selection_sort.c`
- `bubble_sort.c`
- `binary_search.c`

## Notes

These implementations were written manually as a learning exercise rather than copied from external sources.

The focus was understanding algorithm logic, array traversal, comparisons, swapping values, and debugging common mistakes.
