# 01 — C Basics

Small standard C programs written while practicing early CS50 C concepts without using the CS50 library.

## Programs

- `hello.c` — prints hello world
- `name.c` — asks for a name using `fgets`
- `compare.c` — compares two integers
- `password.c` — simple password check using `strcmp`

## Concepts practiced

- `#include`
- `int main(void)`
- `printf`
- `scanf`
- `char` arrays
- `strcmp`
- `if / else if / else`
- integer comparison
- basic string input
- compiling with `make`

## Build and run

```bash
make hello
./hello

make name
./name

make compare
./compare

make password
./password
```

## Notes

These programs use standard C instead of CS50 helper functions such as get_int, get_string, and string.
